PAdding

Flex properties


Flex

Flex is a container which is used to expand items to fill available free space or shrinks them to prevent overflow.

Flexbox

Flex is a layout model in CSS. It allows responsive elements within a container to be automatically arranged depending upon screen size.

Flex-wrap

Flex Wrap specifies whether the flexible items should wrap or not. The default value is nowrap.

Flex-grow

Flex Grow specifies how much the item will grow relative to the rest of the flexible items inside the same container.

Flex-shrink

Flex Shrink CSS property sets the flex shrink factor of a flex item. If the size of all flex items is larger than the flex container,items shrink to fit according to flex-shrink.

Flex-direction

Flex Direction specifies the direction of the flexible items either row or coloumn.

Flex-flow

Flex Flow is the shorthand property for flex-direction and flex-wrap.

Flex-basis

Flex Basis specifies the initial length of a flexible item.

Inline-Flex

Inline Flex displays an element as an inline-level flex container.

Justify-content

Justify Content aligns the flexible container's items when the items do not use all available space on the main-axis.



GRID PROPERTIES



GRID

grid is a grid-based layout model with rows and columns which is used to design webpages without using float and positioning. In Cascading Style Sheets, CSS grid layout or CSS grid creates complex responsive web design grid layouts more easily and consistently across browsers. Historically, there have been other methods for controlling web page layout methods, such as tables, floats, and more recently, CSS Flexible Box Layout. Example, 
                            .grid-container {
                                display: grid;
                              }
All direct children of the grid container automatically become grid items. You can adjust the gap size by using one of the following properties:
  • column-gap
  • row-gap
  • gap
Example for column gap, 
                            .grid-container {
                                display: grid;
                                column-gap: 50px;
                              }
The lines between columns are called column lines. The lines between rows are called row lines.

Grid-template-rows

Grid-template-rows used to specify the no.of rows in a grid layout.

Grid-template-columns

Grid template columns used to specify the no.of columns in a grid layout.

Grid-area

Grid-area is used to specifies the grid item's size and location in a grid layout.

Grid-row

Grid-row is used to specifies the grid item's size and location in a grid row.

Grid-column

Grid-column is used to specifies the grid item's size and location in a grid column.
