CSS Display - Inline , block , inline-block , none , flex


Display: it is a property in CSS which is used to Display elements on the screen.
Block:it occupy the complete width of the screen and always start in new line.
inline:it only occupies the height that only takes the content. it avoids height and width .
Inline-Blockit is a combination of both inline and block.
none:removes the entire content with the skeleton.
note:Visiblity : hidden it hides the content from the screen , but it occupies the space.
Flex:it is one-dimensional property in which we can adjust row and coloumn
